Buenas!

Esta solucion contiene el generador y editor del archivo de credenciales validas. 

	Si desea usarlo, dentro del global.asax podra encontrarlo comentado.
	
Ademas, el path de dicho archivo se maneja de manera tal que este sera donde este alojado
el base directory del proyecto. Y dentro del archivo de configuracion podra modificar el
nombre del archivo de credenciales, dentro de la key "Path" en "appconfig".
